# PhonePay Data Analysis | Power BI Dashboard

## Project Description
This project focuses on analyzing PhonePay transaction data by cleaning raw data, removing duplicates, and building an interactive Power BI dashboard to derive business insights.

## Tools Used
- Power BI
- Microsoft Excel

## Data Cleaning Steps
- Removed duplicate records
- Handled missing values
- Standardized column formats

## Dashboard Highlights
- Total transaction amount analysis
- State-wise transaction performance
- Transaction type comparison
- Year-wise growth trends with filters
